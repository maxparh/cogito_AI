# cogito_AI
ИИ-ассистент для помощи студентам.

Реализованные функции:
- Просмотр расписания
- Помощь при подготовке к экзаменам
- Анти-GPT система
- Советы по научным работам

## Установка
Для развертывания на сервере клонировать репозиторий, после чего запустить файл docker-compose.yaml на сервере с графическим ускорителем не ниже rtx 2060, а также установленным docker.
Рекомендуемая система ubuntu 22.04 или 24.04.
